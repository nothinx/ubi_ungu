01
🛢️

Minyak yang Cukup

Langkah pertama yang krusial. Pastikan minyak goreng menutupi ubi ungu (deep fry level). Jangan pelit minyak agar matang merata.

💬 "tuang minyak, jgn terlalu dikit"
02
🔥

Pengaturan Suhu

Suhu sempurna untuk isi yang lumer! Panaskan minyak dengan api sedang (kecenderungan kecil). Jangan terlalu panas di awal.

💬 "pakai api sedang cenderung kecil aja"
03
🌀

Memasukkan Ubi

Setelah minyak panas, kecilkan api kompor sepenuhnya. Perlahan, masukkan ubi ungu lumer untuk proses pematangan perlahan.

💬 "trs klo dah panas, cemplungin dgn kondisi api kecil"
04
🔄

Pemutaran & Penyajian

Tekstur renyah merata adalah kuncinya. Jangan lupa membalikkan ubi secara berkala. Sajikan saat masih hangat untuk sensasi lumer.

💬 "bolak balik jangan lupa" ... "kalau udah kecoklatan, angkat dan makann"
