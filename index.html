Panduan Menggoreng Ubi Ungu Lumer

1. Persiapan Minyak

Tuangkan minyak goreng ke dalam wajan. Pastikan volume minyak cukup banyak agar ubi dapat terendam dengan baik (tidak terlalu sedikit).

2. Pengaturan Suhu Awal

Panaskan minyak menggunakan tingkat api sedang yang cenderung kecil.

3. Proses Menggoreng

Setelah minyak dipastikan panas, masukkan ubi ungu lumer. Segera atur kompor ke kondisi api kecil.

4. Pematangan Merata

Balik ubi secara berkala selama proses menggoreng agar matang secara merata di semua sisi.

5. Penyajian

Apabila warna ubi sudah berubah menjadi kecoklatan, angkat dan tiriskan. Ubi ungu lumer siap untuk dinikmati.
